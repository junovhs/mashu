Browser-based code context packing

Turn repositories into clean text for ChatGPT and Claude.

MashuPack helps you select exact files or subsystems from a local codebase and compile them into one structured text file for browser AI workflows. It is built for planning, debugging, architecture review, and codebase understanding without messy copy-pasting or file-limit friction.

  • Select exact context. Choose only the files and folders you want the model to see.
  • Export one clean text file. Preserve structure while avoiding binary junk and scattered uploads.
  • Keep code local. MashuPack runs in the browser, with no backend, account, or repo upload.
Read the docs Open MashuPack

MashuPack is for people who still use ChatGPT, Claude, and similar web UIs for long-form software planning, research, and debugging. When file-count limits or upload friction get in the way, a single structured text file is the one format every model can reliably accept.

Instead of manually packing small groups of source files every time you want to discuss a subsystem or ask a high-level question about a codebase, MashuPack gives you a visual browser-based interface for curating the right slice of context and exporting it cleanly.
